The Open Office Server Daemon FAQ


I'm running JUnit Tests with Eclipse, and when I kill them, the daemon is not notified

It seems that the eclipse is not releasing everything when proceeding this way. Try to take it into account and to test other ways (tests without JUnit, or killing JUnit another way).

I have problems when trying to run daemon after running once with ant run

It is important to kill the daemon using ant stop when using ant run. When you launch ant run, it forks a new jvm with soffice as a child of this jvm. When you kill the ant process, the daemon jvm is still alive. You perhaps tried to kill all open office processes, but this daemon jvm will still be running behind.
